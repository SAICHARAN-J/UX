Fitt's Law

Overview

Time taken to complete a pointing task is a function of size of the target, distance to the target and pointing device.

Time Taken (T)

Time taken to complete the pointing task.

Distance (D)

Distance to the pointing object with respect to the pointing device.

Size (W)

It is the size of the pointing object on a particular platform.

Slope (b) and Intercept (a)

a and b are the properties of the pointing device.

F-Fitts

To deal with Accuracy we use F-Fitts law (Fat Finger component + Fitt's law )

Design Principles

Visibility

Wide Viewable Interface with each and every object on the eye.

Eg: M S Word, Windows 10

Mappings

Relationship between the Actions.

Eg: Gas Stove, Utensil Lids

Feedback

Response for a corresponding user action.

Eg: Keyboard, Dialling pad

Constraint

Limited resources and options.

Eg: E-Commerce-Quantity tab, Duolingo selection frame
